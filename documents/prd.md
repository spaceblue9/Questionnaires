# PRD
## Project Name
ระบบประเมินความพึงพอใจในการสอน
## Project Type
internal-tool
## Goal
สร้างเว็บแอปที่ช่วยในการทำงาน เนื่องจาก ผมมีความจำเป็นต้องทำการสอน หรือ แชร์ Knowledge ให้กับทีม โดยทำการสอนทุกอาทิตย์ ครั้งละ 1 ชั่วโมง ผมอยากได้ Web App สำหรับ Feedback การสอนและ Share Material การสอน การประเมินการพึงพอใจ โดยการเก็บข้อมูลไว้ใน Google Sheets เพื่อสรุปผลในปลายปี และมีหน้า Dashboard ในการวิเคราะห์ ข้อมูลในแต่ละครั้งที่สอน หรือดูภาพรวมการสอนทั้งหมด และสามารถมีหน้า Config การใช้ Google Sheet
## Target Users
ผู้สอนทำหน้าที่ ตั้งข้อคำถามในการประเมินความพึงพอใจ ของวิชาที่ตนเองสอน , ผู้เรียนทำหน้าหน้าที่ ประเมินตามหัวข้อที่ผู้สอนกำหนด, ผู้ดูแลระบบ ทำหน้าที่เพิ่ม User และกำหนด Role ในการใช้งาน
## Core Features
- 1. ผมมีความจำเป็นต้องทำการสอน หรือ แชร์ Knowledge ให้กับทีม โดยทำการสอนทุกอาทิตย์ ครั้งละ 1 ชั่วโมง ผมอยากได้ Web App สำหรับ Feedback การสอน
- 2. Share Material การสอน Google Drive
- 3. การประเมินการพึงพอใจ โดยการเก็บข้อมูลไว้ใน Google Sheets เพื่อสรุปผลในปลายปี
- 4. มีหน้า Dashboard ในการวิเคราะห์ ข้อมูลในแต่ละครั้งที่สอน
- 5. ดูภาพรวมการสอนทั้งหมด
- 6. มีหน้า Config การใช้ Google Sheet สำหรับ Admin
- 7. ผู้สอน จะต้องมีหน้า login เพื่อสร้าง Questionnaire Online เพื่อประเมินความพึงพอใจ สำหรับหัวข้ออบรมนั้น
- 8. ผู้อบรม ไม่ต้องมี login แต่ต้องมี Code สำหรับ วิชาการสอนครั้งนั้น ซึ่งผู้สอนจะเป็นคนสร้าง Code สำหรับ วิชานั้น
- 9. admin จะเป็นคน Config เพื่อเชื่อมต่อ Google Drive และ Google Sheets
- 10. สามารถไปรันบน netlify
## User Flow
admin ทำหน้าที่ Config ว่าใครเป็นผู้สอน และสอนวิชาอะไรบ้าง และกำหนด Role ส่วนผู้สอนจะทำการเข้ารายวิชาที่ตนเองสอน และสร้างข้อคำถาม ในการให้ผูเรียนประเมินความพึงใจ และสร้าง link ให้ผู้เรียนประเมินความพึงพอใจ และยกเลิก link หรือตั้งเวลาหมดอายุ ของ link รวมทั้งมีหน้า Dashboard ในการดูผลการประเมินในแต่ละรายวิชา ส่วนผู้ประเมิน จะไม่ต้อง login แต่ต้องได้ link และรหัสผ่าน เพื่อประเมินความพึงพอใจ ครับ
## Success Criteria
อาจารย์ผู้สอนสามารถสร้างแบบสอบถามความพึงพอใจ และมี หน้า Dashboard รายงานผลการประเมิน แต่ละรายวิชา
## Constraints
ใช้ Google sheet เป็น Database ได้หรือไม่มี Database ก็ได้
